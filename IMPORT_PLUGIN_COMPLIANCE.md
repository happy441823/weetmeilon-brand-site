# SWEETMEILON 导入插件合规说明

## 允许处理的来源

只允许处理：

- 自有店铺商品链接
- 已获得授权的商品链接或图片
- 运营人员手动提供的商品资料
- 运营人员手动上传的图片

## 禁止行为

插件不得：

- 抓取竞品详情页素材
- 保存或使用天猫 / 京东 Cookie
- 绕过登录、验证码、风控
- 调用私有接口或逆向接口
- 使用 headless browser 模拟电商平台用户行为
- 高频批量抓取店铺商品
- 伪造销量、价格、库存、评分、评价
- 自动发布未经审核的 AI 内容

## 图片要求

图片入库前必须确认授权。

插件禁止 SVG，限制 MIME 和文件大小，并生成安全文件名。被商品、文章或页面引用的素材不得删除。

## 成人用品内容要求

内容必须专业、克制、合规：

- 不涉未成年人
- 不承诺医疗功效
- 不使用低俗表达
- 不虚构销量或评价
- 不使用绝对化高风险词

## 第一阶段边界

第一阶段只提供插件界面和基础导入能力。

`CMS_AI_GENERATION_ENABLE=false`  
`CMS_SEO_AUTO_SUBMIT=false`  
`CMS_SEO_INDEXNOW_ENABLE=false`  
`CMS_PUBLIC_D1_READS=false`
