# SWEETMEILON 后台 CMS 架构

本后台将官网内容分为商品、文章、页面、FAQ、媒体、导航、页脚、SEO 设置、重定向、管理员、操作日志和定时发布任务。

数据保存位置：

- Cloudflare D1：保存所有结构化内容和操作日志。
- Cloudflare R2：保存商品图、文章图、品牌素材和 OG 图。
- Cloudflare Access：保护 `/admin/*` 和 `/api/admin/*`。

公开官网读取规则不变：草稿、待审核、未到发布时间的内容不会公开展示。当前代码已提供 D1 表结构、后台页面和后台 API，公开页面切换到 D1 正式数据源需要在开发 D1 完成迁移验收后再开启。
