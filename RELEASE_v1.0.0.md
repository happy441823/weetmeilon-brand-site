## 本版规则

- 只公开 `active` 且人工审核通过的商品。
- `upcoming` 商品只显示新品预告，不显示购买按钮。
- 未审核商品不进入首页、产品中心、分类页、详情页、sitemap。
- 官网不展示价格、销量、付款人数、抓取来源和审核状态。
