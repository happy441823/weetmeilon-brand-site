# DESIGN.md

## 视觉定位

蜜女郎官网不是普通商城首页，而是「高级品牌落地页 + 官方渠道成交引导页」。视觉要比平台店铺首页更克制、更品牌化，同时保留年轻化、二次元、轻赛博和大字报式电商记忆点。

## 色彩

- 主背景：`#100019`、`#13001F`、`#1D0630`
- 深紫层级：`#28103D`、`#32164E`、`#4A1F72`
- 薄荷绿：`#90FFE7`
- 浅青绿：`#53F0D0`
- 辅助渐变：`linear-gradient(135deg, #90FFE7, #53F0D0, #BCA4FF)`
- 正文浅色：`#F7ECFF`
- 次级文字：带透明度的浅紫白

## 字体

- 系统无衬线字体，优先保证中文阅读清晰。
- 首页 H1 使用超大标题，形成「真实质感，从材质开始」的强记忆点。
- 正文保持 14-18px，行高 1.7-2，避免压迫感。

## 按钮

统一购买按钮文案：

- 前往天猫旗舰店
- 前往京东旗舰店
- 查看天猫同款
- 查看京东同款

样式：

- 胶囊圆角。
- 薄荷绿/浅青绿渐变。
- 深紫文字，强点击感。
- 点击统一触发 `click_tmall_button`。

## 图片风格

当前首页产品图读取 `public/images/products/`，品牌细节图读取 `public/images/brand/`，后续替换真实图片时建议：

- 产品质感微距。
- 材质表面细节。
- 边缘过渡与柔雾光泽。
- 包装与官方渠道细节。
- 深紫背景与薄荷青绿光效。

避免：

- 低俗、露骨、过度身体化画面。
- 夸张姿态或强刺激构图。
- 虚假认证、检测报告、销量截图。
- 医疗功效暗示。

## 页面风格

- 手机端优先，桌面端扩展为双栏或三栏。
- 深色背景搭配轻赛博网格、柔和径向光效。
- 卡片用于产品、FAQ、文章列表，不做复杂嵌套。
- 首屏强调品牌名、官方身份、材质科技和官方渠道跳转。

## 组件规范

- `SiteHeader`：Logo + 品牌名 + 主导航 + 官方渠道购买菜单。
- `AgeGate`：18+ 年龄确认。
- `StoreButtons`：全站统一官方渠道购买跳转。
- `ProductCard`：产品卡片，读取 `products.ts`。
- `SectionHeader`：统一栏目标题。
- `TrustStrip`：官方、隐私、材质、清洁四个信任点。
