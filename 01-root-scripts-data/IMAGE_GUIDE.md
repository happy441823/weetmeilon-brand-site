# IMAGE_GUIDE.md

## 产品主图

- 推荐尺寸：1200 x 1200px
- 推荐格式：PNG 透明背景
- 文件大小：单张尽量控制在 500KB 以内
- 存放目录：`public/images/products/`
- 当前首页读取路径：
  - `/images/products/product-01.png`
  - `/images/products/product-02.png`
  - `/images/products/product-03.png`
- 已上架商品建议使用商品 ID 目录，例如：
  - `/images/products/half-body-public-review/cover.png`
  - `/images/products/half-body-public-review/gallery-01.webp`
- 构图建议：产品主体居中，四周保留安全留白，避免边缘被按钮、标签或圆角容器压住。
- 画面建议：深紫背景、薄荷青绿轻光效、克制材质细节；产品主体清晰，不使用重度模糊。

## 品牌细节图

- 推荐尺寸：1200 x 900px
- 推荐格式：JPG、WebP 或 PNG
- 存放目录：`public/images/brand/`
- 当前首页读取路径：
  - `/images/brand/material-detail.png`
  - `/images/brand/package.png`
  - `/images/brand/privacy-shipping.png`
  - `/images/brand/care-storage.png`
- 构图重点：材质、包装、隐私发货和清洁收纳细节。
- 表达边界：不使用露骨画面，不展示无法证明的认证、检测报告、销量或功效承诺。

## 产品详情图

- 建议内容：表面纹理、边缘过渡、柔雾质感、包装信息、清洁收纳细节。
- 避免方向：过度暴露、夸张姿态、医疗暗示、虚假检测认证图。
- alt 文案建议：围绕材质、质感、清洁、包装、官方渠道等客观信息，不写露骨关键词。

## 首页首屏图

- 推荐尺寸：1600 x 1200px
- 推荐格式：WebP 或 PNG
- 构图建议：产品主体靠右或居中偏右，左侧预留标题文字空间。
- 视觉建议：深紫色背景、薄荷绿轻光效、材质细节清晰，避免重度模糊和过亮遮罩。
